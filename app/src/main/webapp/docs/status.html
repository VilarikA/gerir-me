Preferencialmente não exclua, use o Status.

Quando você cria uma pessoa, colaborador, serviço, produto, etc; tão logo este elemento seja vinculado a uma transação o não permite mais que ele seja excluído.

O campo de status, quando preenchido com "inativo" serve justamente para que você informe ao que não faz mais aquele, serviço, não vende mais aquele produto, não trabalha mais com aquele colaborador, ou não mais se relaciona com aquela pessoa.

Quando marcado como intativo o elemento deixa de aparecer como opção de escolha para novas transações, mas aparece em todas as transações históricas de que tenha participado.

Importante ressaltar que a qualquer momento você pode alterar o status de um item inativo para torná-lo ativo novamente. Para tanto basta marcar a opção mostrar todos nas opções de pesquisa acessar o item inativo e alterar o status para "ativo" novamente.
