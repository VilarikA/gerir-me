Agendamentos periódicos/recorrentes.

Para fazer agendamentos periódicos, use a tecla de atalho Y, ou vá na opção clientes no menu superior e clique no quarto ícone localizado na parte superior esquerda da tela, uma ampulheta.

Então basta preencher os campos na tela: Cliente, profissional e serviço a ser feito. Escolha a próxima data com o dia da semana desejado e uma data para término do horário fixo: seis meses, um ano, dois anos, etc.

Quantidade Limite

Informando um número diferente de zero na quantidade limite, quantidade de agendamentos erá limitada por este número, mas para isso a data fim deve ser suficiente para comportar este agendamentos, caso contrário a data fim definirá o limite.

Hora

Coloque a hora de inicio, se a hora de fim for omitida será considerada a duração do serviço, caso ela seja informada este valor será acatado.

Agendar

Clique no botão gerar. O cria uma tabela abaixo do formulário com a projeção dos atendimentos. Você pode excluir um ou outro agendamento se quiser. Você pode ainda, antes de gravar, alterar o profissional e/ou alterar o serviço e/ou as demais configurações e gerar novamente, a tabela será acrescida destes novos agendamentos.

Quando os agendamentos estiverem certos clique no botão gravar.

Caso seja necessário fazer alterações na programação, clique no botão limpar e refaça a programação antes de gravar.

Observe que se você clicar no botão gerar duas ou mais vezes o vai dobrando os atendimentos na tabela, caso isso ocorra clique no botão limpar e refaça a programação antes de gravar.

Excluir

Para excluir lançamentos periódicos, utilize o relatório de Resumo de Atendimentos (atalho T). Informe o , o , o intervalo de datas e clique no botão Buscar para ver os lançamentos que serão excluídos, posteriormente clique no botão Excluir.
