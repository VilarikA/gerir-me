A agenda é um dos componentes chave do .

Para fazer um agendamento simplesmente clique no /horário desejado, selecione um serviço e clique no botão adicionar. Use a tecla (ESC) ou clique no (x) para fechar a janela de agendamento e retornar para a visualização da agenda.

Para alterar detalhes de um agendamento dê um clique duplo sobre o agendamento existente. Você pode adicionar novos serviços, excluir serviços ou altetar inicio, fim e/ou duração dos mesmos.

Para alterar apenas a duração de um agendamento posicione o cursor na linha inferior do agendamento e arraste para cima (diminuir a duração) ou para baixo (aumentar a duração).

Para deslocar um agendamento no tempo ou para outro , clique na barra superior, de cor destacada, do agendamento e arraste para cima ou para baixo, alterando o horário, ou arraste para o lado, trocando o agendamento de um para outro.

Para deslocar um agendamento para uma outra data mantendo o mesmo , selecione primeiro o na parte superior da agenda, selecione uma opção de visualização, 3, 5, ou 7 dias, clique na barra superior de cor destacada e arraste o atendimento para o dia desejado.
