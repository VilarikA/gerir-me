Limpando o cache do navegador.

A medida que você navega na internet ou executa uma aplicação web como o , o google chrome e também os demais navegadores armazenam numa memória local chamada cache, imagens e scripts das páginas que você acessa. O objetivo por trás disso é deixar a sua navegação mais rápida, pois quando você retorna àquela página o navegafor não precisa acessar novamente o servidor através da internet.

Quando uma versão nova da aplicação é atualizada, páginas, scripts e imagens novos e/ou modificados são colocados no servidor, o que pode causar uma inconsistência entre o que está lá e o que seu navegador armazenou no cache local.
