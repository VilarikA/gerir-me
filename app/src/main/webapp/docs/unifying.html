Unificando clientes no .

Para unificar clientes vá na opção clientes no menu superior e clique no segundo ícone no alto a esquerda, após abrir a janela você vai colocar no primeiro campo o cliente que vai "sumir" e no segundo campo o cliente que vai "ficar", todo o histórico do primeiro cliente se houver, será transferido para o segundo, e o primeiro cliente será excluído após a unificação.

Quantoas informações do próprio cadastro acontece o seguinte: se por exemplo, o celular do primeiro estiver prenchido e o celular do segundo estiver vazio, o copia o celular do primeiro para o segundo, mas se o segundo já tiver um celular, este celular será mantido e se o celular do primeiro for diferente do celular do segundo ele será perdido na exclusão.

Se houver uma diferença no nome dos dois clientes as serem unificados, fica fácil você selecionar quem vai "sumir" e quem vai "ficar", se os nomes forem idênticos você pode verificar o id de cada um que é o numero interno dado pelo sistema. Este número aparece quando você lista os clientes. Então você pode escolher os clientes na unificação pelo id na caixa de texto que vem a esquerda da caixa para digitar o nome.

Profissionais

Todo profissional, também aparece como cliente, sem ter que ter o seu cadastro duplicado, mas isso pode acontecer. Então a unificação pode ser usada para profissionais e clientes e mesmo para dois profissionais. Importante ressaltar que se dois profissionais forem ser unificados, os horários, serviços e unidades relacionados ao profissional que vai "sumir", devem ser excluídos antes, pois não podem ser duplicados no profissional que "ficar".
